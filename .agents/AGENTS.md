# Project Core Principles (Kerala Government)

- **Uncompromising Quality & Security:** Every feature must be built with maximum security and safety as top priorities. No compromise.
- **20-Year Maintainability:** The code foundation must follow clean architecture, best practices, and thorough documentation so that developers decades from now can easily understand and maintain it.
- **Zero Broken Code:** No quick hacks, messy workarounds, or broken code. Every piece of code must be robust, fully functional, and well-tested before being finalized.
- **Extreme Scalability & Performance:** The code must be optimized to handle enormous amounts of stress and traffic without degradation.
- **Absolute Responsiveness:** Full feature parity and flawless adaptable UI across all screen sizes for Web, iOS, and Android.
- **Best-in-Class Advanced Architecture:** Utilize modern, state-of-the-art scalable architectural patterns suitable for enterprise-grade Flutter applications.
- **Universal Device Support & Optimization:** Must run flawlessly on all hardware without restriction, from extremely low-end devices to high-end flagships. Memory, CPU, and battery usage must be heavily optimized.
